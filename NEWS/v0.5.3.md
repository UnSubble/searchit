# Searchit v0.5.3 - Trustworthy Updates

Searchit v0.5.3 is a bugfix and usability release focused exclusively on **Trustworthy Updates**.

## Why was this release created?

We discovered that our update mechanism (`searchit update --install`) could be misleading for some users. While it successfully installed the newest version of Searchit in your standard Go binary path, users who originally installed Searchit through their system package manager or to a root directory (like `/usr/bin/searchit`) were not actually seeing the update take effect when running the tool.

## What problem does it solve?

This release introduces absolute transparency around updates. A successful installation no longer blindly equates to a successful update. Searchit now recognizes the difference between installing a new binary and successfully replacing the *active* executable that you're running. 

## How does Searchit behave differently now?

- **Active Executable Validation**: Searchit dynamically tracks where your currently running version lives and cross-references it with where updates are installed.
- **Multiple Binary Detection**: The tool warns you if you have competing versions of Searchit across your `PATH`.
- **System Installation Guidance**: If Searchit detects that you are running a system-installed version (e.g., via `/usr/bin/`), it will gracefully finish downloading the update, warn you that the active executable remains outdated, and generate the exact, personalized `sudo install` command you need to fix it safely.
- **No Assumptions**: Searchit will no longer attempt to guess or forcefully install updates if your installation environment appears exotic or unsafe.

## What should users expect when updating?

If you installed Searchit via standard `go install`, updates will behave seamlessly exactly as before! If you use a system installation, you will now see clear, actionable instructions on how to finalize your update. Lastly, `searchit doctor` now includes dedicated installation and PATH diagnostics to ensure your environment stays healthy.
