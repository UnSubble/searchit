# RELEASE v0.5.2

### FIXED

- Fixed interactive terminal raw mode ignoring Ctrl+C inputs (`\x03`) due to disabled ISIG flags.
- Fixed case-sensitivity bug where Shift+key inputs (e.g., 'A', 'P') were ignored during scans.
- Corrected progress UI legend to display accurate target stop (`q`) and global abort (`a`) controls.
- Added comprehensive unit and integration tests verifying raw terminal signals and aborted state combinations.

> [!NOTE]
> v0.5.2 is a critical patch release addressing regressions in the global abort signaling path and console interactive controls.
